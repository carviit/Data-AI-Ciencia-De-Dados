Comandos GIT

--> git diff (Verificar qual arquivo foi alterado e qual a alteração)

--> git log (Verifica histórico de commits em log)

--> git log --oneline (Histórico de log mais simples de visualização)

--> git commit -a -m "nome do commit" (Pula estágio staging do git)

--> git help comando (Documentação do comando)

--> git restore --staged nomearquivo.jpg (Recupera do staged).

--> git restore nomearquivo.jpg (Recupera arquivo)

--> git mv nomearquivo.jpg nomenovoarquivo.jpg (Renomear arquivo)

--> git -amend (Altera o último commit)

--> git log -p (Mostra log com detalhes de o que foi alterado em cada commit)

--> git reset --hard codigocommit (Desfaz o ultimo commit e retorna para o penultimo ou selecionado)

--> git config alias.nomeatalho "comando se tiver espaço" se não não tem aspas.

--> git branch nomebranch (Cria branch)

--> git branch (Lista todas as branch)

--> git switch nomebranch (Altera para a branch)

--> git branch -d nomebranch (Deletar branch)
